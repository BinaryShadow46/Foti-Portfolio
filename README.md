Changing Personal Information

Edit the following sections in index.html:

1. Name & Headline: Lines with "Fotinati Ndele" and "Bridging Medicine & Technology"
2. Bio Text: Paragraph in the hero section
3. Project Details: Update project titles, descriptions, and technologies

Changing Colors

Modify the CSS variables in the <style> section:

```css
:root {
    --primary-blue: #0A192F;      /* Main background */
    --secondary-blue: #112240;    /* Card backgrounds */
    --accent-blue: #64FFDA;       /* Accent color */
    --text-light: #E6F1FF;        /* Primary text */
    --text-gray: #8892B0;         /* Secondary text */
}
```
